# Event-Registration
This is a simple project done as part of an internship at [MushStack, Kallampally](https://maps.app.goo.gl/d3zXzxg4niMKxvgP8). This is a small project with few features. From the project me & my friends learned Mongodb, Express, React & NodeJs.

## What this project does?
It simply help to manage event registration. (Just try & understand)

## How to run this application?
At first this is web application. It have separate frontend & backend code.

### Steps to run:
1. Install [NodeJs](https://nodejs.org/en/download), [MongoDB](https://www.mongodb.com/try/download/community) & [Git](https://git-scm.com/install/)
2. Open terminal & go to a work folder
3. Run these commands to setup project on your machine:
```Terminal
git clone https://github.com/Sajeevbkk/Event-Registration.git
cd Event-Registration
```
4. Run these commands to start backend
```Terminal
cd backend
npm install
npm start
```
5. Run these commands to start frontend
```Terminal
cd ..
cd frontend
npm install
npm run dev
```
Now go open [LocalHost at port 5173](http://localhost:5173/)

# Hurray!! Your run the application.
